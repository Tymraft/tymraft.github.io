GitHub access token

To upload files, enter a GitHub Personal Access Token with repo write access. It's stored in memory only and cleared when you close this tab.

Generate one at github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens (scope: Contents: read & write).
